Humans learned from YouTube for years.Now let your agent do the same.

How it works

            
One channel in. A compounding skill library out.

          

          

            
01
Drop a channel plus intent
Drop a YouTube channel URL and what you want your agent to learn. SkillForge handles the rest.

            
02
Agent scans and proposes skills
SkillForge ranks the best videos, extracts the relevant frameworks, then suggests reusable skills.

            
03
Skills saved to memory permanently
Ship the approved skills straight into agent memory so future tasks start smarter.

          

        


        

          
Use cases
Built for operators who want leverage, not more content.

          

            
Learn from any expert
Point at any YouTube channel, extract the sharpest frameworks, and turn them into agent skills your stack can reuse.

            
Intent-first intelligence
Not a dump of everything. SkillForge filters for exactly what matters to your next task.

            
Compounds over time
Watch the same creator over time and skills compound — new videos merge in, nothing overwrites. Your agent gets sharper with every session.

            
Creator-organized library
Skills live at ~/.skillforge/library/@creator/topic.skill.md — full attribution, provenance, and source videos tracked in every file.

            
MCP server built in
Run skillforge serve to expose your library as an MCP server. OpenClaw, Claude Desktop, and any MCP-compatible agent can query skills on demand.

when_to_use:
  - Drafting funnel strategy
  - Writing offers and objections handling
  - Reviewing qualification and close steps

instructions:
  - Lead with value-stack, urgency, and proof
  - Remove friction between lead capture and close
  - Handle objections with diagnosis before pitch
  - Use qualification criteria before escalation

memory:
  persistence: permanent
  approved_by: operator

4 steps. Under 60 seconds.

        
Get your agent learning right now

        
    
          
  1. 
            Install yt-dlp (transcript engine)
            
    
              brew install yt-dlp
              
            
    
            
    Linux/Windows: pip install yt-dlp
    
          
    2. 
          
  2. 
            Install SkillForge
            
    
              npm install -g skillforge
              
            
    
            
    Also set ANTHROPIC_API_KEY or OPENAI_API_KEY in your environment.
    
          
    3. 
          
  3. 
            Watch a video — preview before saving
            
    
              skillforge watch https://youtube.com/watch?v=VIDEO_ID
              
            
    
            
    SkillForge previews the skill first — you approve before anything saves to your library.
    
          
    4. 
          
  4. 
            Query your library — agents call this
            
    
              skillforge recall "sales funnel"
              
            
    
            
    Or run skillforge serve to expose your library as an MCP server — any compatible agent can query it on demand. Skills are saved at ~/.skillforge/library/@creator/topic.skill.md.
