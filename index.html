How it works

            
One channel in. One durable skill out.

          

          

            
01
Drop a channel plus intent
Pass a creator handle and the exact problem your agent needs to solve next.

            
02
Agent scans and proposes skills
SkillForge ranks the best videos, extracts the relevant frameworks, then suggests reusable skills.

            
03
Skills saved to memory permanently
Ship the approved skills straight into agent memory so future tasks start smarter.

          

        


        

          
Use cases
Built for operators who want leverage, not more content.

          

            
Learn from any expert
Point at any YouTube channel, extract the sharpest frameworks, and turn them into agent skills.

            
Intent-first intelligence
Not a dump of everything. SkillForge filters for exactly what matters to your next task.

            
Compounds over time
Every saved skill stacks into long-term memory, so your agent gets stronger each session.

# SKILL.md
name: sales-funnel-and-closing-framework
source: youtube/@AlexHormozi
intent: build a sales funnel

when_to_use:
  - Drafting funnel strategy
  - Writing offers and objections handling
  - Reviewing qualification and close steps

instructions:
  - Lead with value-stack, urgency, and proof
  - Remove friction between lead capture and close
  - Handle objections with diagnosis before pitch
  - Use qualification criteria before escalation

memory:
  persistence: permanent
  approved_by: operator

  2. 
            Point it at a YouTube channel
            
    
              skillforge scan @channelname --intent "what you want to learn"
